Tab Strip

When building a mobile web app, there is a time we need to build a tab strip to contain our menu. When creating a tab strip, please provide and title that represent the content of the link. For the grid system, you can use .tabspan-n classes, with the n is a number between 1 - 10. It means when the you use .tabspan-4, the width of the span will be 100/4 % or 20%.
