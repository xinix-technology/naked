Naked CSS

Icons

Naked CSS use Entypo as default Icon.

You can place the Icons just about anywhere using the CSS Prefix xn and the icon's name. The Icons is designed to be used with inline elements (we like the <i> tag for brevity, but using a <span> is more semantically correct).
